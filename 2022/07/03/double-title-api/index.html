自实现双标题接口

如果您想要实现自己的双标题接口，则需要了解以下参数：

参数名称 说明 位置 PHP示例
HTTP_GGDEV_CDKEY 授权码 请求头 $_SERVER[‘HTTP_GGDEV_CDKEY’]
HTTP_GGDEV_ACTIVATE_DOMAIN 授权域名 请求头 $_SERVER[‘HTTP_GGDEV_ACTIVATE_DOMAIN’]
title 文章标题 body $_POST[‘title’]

上面的3个参数，在插件请求时会携带过来，您可以根据以上参数来做接口限制。

接口返回数据参数说明：

参数名称 说明
data 副标题数组，是一维数组（数组里的值为字符串）
status 有副标题，设为1，没有副标题设为0
msg 自定义返回信息
